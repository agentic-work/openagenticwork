# @agentic-work/agenticode

**AgentiCode** - AI-powered code assistant CLI.

Part of the [AgenticWork](https://agenticwork.io) enterprise AI platform.

## Features

- Interactive terminal UI with rich formatting
- Multi-provider LLM support via @agentic-work/sdk
- File operations, shell execution, and web search tools
- MCP (Model Context Protocol) integration
- Session persistence and context management

## Installation

```bash
npm install -g @agentic-work/agenticode
```

## Usage

```bash
agenticode
```

Or with options:

```bash
agenticode --model gpt-4 --ui ink
```
