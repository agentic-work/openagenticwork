# Contributing to OpenAgenticWork

Thank you for your interest in contributing to OpenAgenticWork! We welcome community feedback, bug reports, and feature suggestions.

## How to Contribute

### Reporting Bugs

1. Check if the issue already exists in [GitHub Issues](https://github.com/agentic-work/openagenticwork/issues)
2. If not, create a new issue with:
   - Clear description of the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment details (OS, Docker version, etc.)

### Suggesting Features

1. Open a new issue with the `enhancement` label
2. Describe the feature and its use case
3. Explain why it would benefit the community

### Submitting Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes following our code style
4. Ensure all tests pass
5. Submit a PR with a clear description

**Note:** All PRs are reviewed by the Agenticwork team before merging. External contributors cannot merge directly.

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on the code, not the person

## Contact

For questions or support, reach out to: support@agenticwork.io
